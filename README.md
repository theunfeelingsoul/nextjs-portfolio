# Next.js Portfolio（ポートフォリオ）

A clean, multilingual portfolio built with Next.js (App Router).  
Designed to be simple, fast, and focused on showcasing projects.

## 🌐 Live Demo（デモ）

https://nextjs-portfolio-fxa3z4mex-angel-beats-projects.vercel.app/en

## 🌐 Preview

![Preview](/preview.png)

## ✨ Features（特徴）

- 🌍 Internationalization (English / Japanese)
- 🧭 App Router structure (Next.js 15)
- 📁 Dynamic project pages
- 📬 Contact section with social links
- ⚡ Fast and responsive layout

## 🛠️ Tech Stack（技術スタック）

- Next.js
- React
- JavaScript (ES6+)
- CSS

## 📁 Project Structure（プロジェクト構成）

```bash
src/
├─ app/
│  ├─ [locale]/
│  │  ├─ page.js
│  │  ├─ about/
│  │  ├─ projects/
│  │  └─ contact/
├─ lib/
│  ├─ getDictionary.js
│  └─ projects.js

```

Clone the repo and install dependencies:

```bash
npm install
npm run dev
```

Then open:

http://localhost:3000
